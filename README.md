# DecodeLabs Tasks

This repository contains all tasks completed during the DecodeLabs Data Science Internship Program.

## Tasks

### Task 1 - Advanced EDA & Feature Engineering

#### Objective

Perform exploratory data analysis and feature engineering on a housing dataset to prepare data for machine learning applications.

#### Key Activities

* Data Exploration
* Missing Value Analysis
* Outlier Detection
* Feature Engineering
* Correlation Analysis
* Data Preprocessing

#### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

---

### Task 2 - Customer Segmentation using K-Means Clustering

#### Objective

Segment customers into meaningful groups based on demographic and spending behavior using unsupervised machine learning techniques.

#### Key Activities

* Data Preprocessing
* Label Encoding
* Feature Scaling
* Principal Component Analysis (PCA)
* Elbow Method
* K-Means Clustering
* Silhouette Score Evaluation
* Customer Persona Analysis

#### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
